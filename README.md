# BuyBye — ShopList

The simplest shopping list app. No account, no backend, no friction.

🌐 **Live:** [www.getbuybye.com](https://www.getbuybye.com)

## Features

- ➕ Add items — auto-categorized (Meat, Produce, Bakery, Dairy, Drinks, Grocery)
- ✅ Check off items — move to "Done" after 1 hour
- 🗑️ Delete — swipe left or long press
- 🇵🇹🇺🇸 Bilingual — Portuguese & English, preference saved locally
- 📱 PWA — installable on iPhone & Android
- 💾 No server — everything in localStorage

## Stack

Single HTML file. Zero dependencies. Zero build step.

## Deploy

Just upload `index.html` + `manifest.json` + icons to any static host.
